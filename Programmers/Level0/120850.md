# 문제 이름
## [Lv.0 - 120850](https://school.programmers.co.kr/learn/courses/30/lessons/120850)

### 문제
- 문자열 my_string이 매개변수로 주어질 때, my_string 안에 있는 숫자만 골라 오름차순 정렬한 리스트를 return 하도록 solution 함수를 작성해보세요.
### 풀이
- 받아온 string 타입의 변수를 이용해 int로 변환할 수 있는지 확인을 한다.
- 코드에 나온 풀이에서는 `IsDigit()` 라는 함수를 사용했는데, ASCII 코드 기준으로 확인하고 10진수인 0~9만 확인할 수 있다.
- 만약 소수 등 수 자체를 확인해야 하는 거라면 `char.IsNumber()`를 사용하면 된다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(string my_string) {
        
        List<int> answer = new List<int>();
        
        foreach(char c in my_string)
        {       
            // char 타입의 변수가 0~9 라는 숫자를 가지고 있는지 확인
            if(char.IsDigit(c))
            {
                // IsDigit은 ASCII 코드로 확인하는 거라 0(ASCII : 48)의 문자를 빼야 일반 숫자로 변환됨
                answer.Add((int)c - '0');
            }
        }
        
        // 오름차순 정렬
        answer.Sort();
        
        // 정답은 int[] 타입이라 타입 변경
        return answer.ToArray();
    }
}
```

### 다른 풀이
- `IsDigit()` 방법이 아닌 정석으로 설명되는 `TryParse()` 풀이는 다음과 같다.
- 코드를 보면 알다시피 string 타입 자체가 정수로 변환 가능한지 확인하는 거라 문제처럼 한 문자씩 확인하는 거라면 조금은 번거로울 수 있다.
- 참고로, 해당 함수의 인수 `out` 없으면 arguments 오류가 뜬다.
``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(string my_string) {
        
        List<int> answer = new List<int>();
        
        // 정수로 타입 변경 가능하면 해당 변수에 값이 변경됨
        int num = 0;
        
        foreach(char c in my_string)
        {       
            // 정수로 타입 변경 가능한지 확인
            if(int.TryParse(c.ToString(), out num))
            {
                answer.Add(num);
            }
        }
        
        // 오름차순 정렬
        answer.Sort();
        
        // 정답은 int[] 타입이라 타입 변경
        return answer.ToArray();
    }
}
```

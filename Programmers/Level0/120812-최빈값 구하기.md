# 최빈값 구하기
### [Lv.0 - 120812](https://school.programmers.co.kr/learn/courses/30/lessons/120812)

## 문제
최빈값은 주어진 값 중에서 가장 자주 나오는 값을 의미합니다.
정수 배열 `array`가 매개변수로 주어질 때, 최빈값을 return 하도록 solution 함수를 완성해보세요. 최빈값이 여러 개면 `-1`을 return 합니다.
## 풀이
- 배열마다의 카운드와 최대 빈도 수를 잘 확인하면 된다.
- 여기서 최대 빈도 수는 같지만 수는 다른 경우가 있기에 이 부분에 대한 검사가 중요하다.

``` C#
using System;

public class Solution {
    public int solution(int[] array) {
        // 길이가 1인 배열도 있기에 길이로 먼저 정답 설정
        int answer = array.Length;
        // 혹시 모를 오름차순 정렬
        Array.Sort(array);
        
        // 가장 많이 보이는 카운트
        int maxCount = 0;
        // 현재 카운트
        int currentCount = 0;
        // 최대 카운트에서 중복이 있는지 확인
        bool isOverlap = false;
        
        for(int i = 0; i < array.Length; i++)
        {
            // 카운트 표시
            currentCount++;
            
            // 마지막 숫자거나 새로운 수로 갱신이 된 상태라면
            if(i+1 == array.Length || array[i] != array[i+1])
            {
                // 최대 빈도 수가 갱신되었을 때
                if(maxCount < currentCount)
                {
                    // 중복해제 및 카운트/숫자 갱신
                    isOverlap = false;
                    maxCount = currentCount;
                    answer = array[i];
                }
                // 최대 빈도 수가 동일할 경우
                else if(maxCount == currentCount)
                {
                    // 중복 발견
                    isOverlap = true;
                }
                // 새로운 수로 갱신되었기에 카운트 초기화
                currentCount = 0;
            }
            
        }
        
        // 중복이 있으면 -1, 없으면 갱신된 최고 숫자
        return (isOverlap) ? -1 : answer;
    }
}
```

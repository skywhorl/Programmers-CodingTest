# 정사각형으로 만들기
### [Lv.0 - 181830](https://school.programmers.co.kr/learn/courses/30/lessons/181830)

## 문제
- 이차원 정수 배열 `arr`이 매개변수로 주어집니다.
- `arr`의 행의 수가 더 많다면 열의 수가 행의 수와 같아지도록 각 행의 끝에 0을 추가하고, 열의 수가 더 많다면 행의 수가 열의 수와 같아지도록 각 열의 끝에 0을 추가한 이차원 배열을 return 하는 solution 함수를 작성해 주세요.
## 풀이
- 배열을 생성할 때 빈 자리는 0이 자동으로 되므로 길이에 맞게 새로운 배열을 생성해 맞춰주면 된다. 

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[,] solution(int[,] arr) {
        // 더 긴 쪽으로 설정
        int length = (arr.GetLength(0) > arr.GetLength(1)) ? arr.GetLength(0) : arr.GetLength(1);
        // 행렬의 길이가 같은 배열 생성
        int[,] answer = new int[length,length];
        
        // 기존의 내용을 넣기, 이외의 빈자리는 자동으로 0이기에 따로 코드로 설정 안 함
        for(int i = 0; i < arr.GetLength(0);  i++)
        {
            for(int j = 0; j < arr.GetLength(1); j++)
            {
                answer[i,j] = arr[i,j];
            }
        }
        
        return answer;
    }
}
```

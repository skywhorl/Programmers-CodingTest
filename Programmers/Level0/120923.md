# 연속된 수의 합 
### [Lv.0 - 120923](https://school.programmers.co.kr/learn/courses/30/lessons/120923)

## 문제
- 연속된 세 개의 정수를 더해 12가 되는 경우는 3, 4, 5입니다.
- 두 정수 `num`과 `total`이 주어집니다.
- 연속된 수 `num`개를 더한 값이 `total`이 될 때, 정수 배열을 오름차순으로 담아 return하도록 solution함수를 완성해보세요.
## 풀이
- 연속된 수들 중에 가장 작은 수를 n이라 하면`num * n`개의 연속된 수와 공차가 1인 `num`개의 수들의 합을 구하면 `total`이 된다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(int num, int total) {
        List<int> answer = new List<int>();
        
        // 연속된 숫자의 가장 작은 값 구하기
        int n = (total - CommonDifferenceSum(num)) / num;
        
        // 시작하는 숫자와 공차를 알았으니 차례로(오름차순) 넣기
        for(int i = 0; i < num; i++)
        {
            answer.Add(n+i);
        }

        return answer.ToArray();
    }
    
    // 연속된 숫자들의 공차들의 합 구하는 함수(공차가 1임)
    private int CommonDifferenceSum(int num)
    {
        int commonSum = 0;
        for(int i = 0; i < num; i++)
        {
            commonSum += i;
        }
        return commonSum;
    }
}
```

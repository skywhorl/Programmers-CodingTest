# 문제 이름
## [Lv.0 - 181832](https://school.programmers.co.kr/learn/courses/30/lessons/181832)

### 문제
- 양의 정수 n이 매개변수로 주어집니다. n × n 배열에 1부터 $$n^2$$ 까지 정수를 인덱스 [0][0]부터 시계방향 나선형으로 배치한 이차원 배열을 return 하는 solution 함수를 작성해 주세요.
### 풀이
- 나선형 방향대로 각각 [좌>하>우>상]으로 나눠 숫자를 채웠다.
- while 조건문을 통해 다음 행/렬의 내용이 0이 아닐 때까지 이동하도록 설정했다.
- 반복문과 배열에서의 Array Out of Bounds 오류를 방지하기 위해 다음 행/렬이 범위 초과인지까지 확인한 후 범위를 올리는 방식을 사용했다.

``` C#
using System;

public class Solution {
    public int[,] solution(int n) {
        int[,] answer = new int[n,n];
        
        // 배열 안에 넣을 값
        int num = 1;
        // 행렬 포인트 설정
        int row = 0;
        int column = 0;
        // 시작점 설정
        answer[0,0] = 1;
        
        // 나선 주기 = (n/2) + (n%2)
        for(int i = 0; i < (n/2) + (n%2); i++)
        {
            // 배열의 다음 내용을 확인한 후 값 설정
            // 오른쪽 이동
            while(column+1 < n && answer[row,column+1] == 0)
            {
                column++;
                num++;
                answer[row,column] = num;
            }
        
            // 아래 이동
            while(row+1 < n && answer[row+1,column] == 0)
            {
                row++;
                num++;
                answer[row,column] = num;
            }   
        
            // 왼쪽 이동
            while(column-1 >= 0 && answer[row,column-1] == 0)
            {
                column--;
                num++;
                answer[row,column] = num;
            }
        
            // 위 이동
            while(row-1 > 0 && answer[row-1,column] == 0)
            {
                row--;
                num++;
                answer[row,column] = num;
            }
        }
        
        return answer;
    }
}
```

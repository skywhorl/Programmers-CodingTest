# OX 퀴즈
### [Lv.0 - 120907](https://school.programmers.co.kr/learn/courses/30/lessons/120907)

## 문제
덧셈, 뺄셈 수식들이 `'X [연산자] Y = Z'` 형태로 들어있는 문자열 배열 `quiz`가 매개변수로 주어집니다. 
수식이 옳다면 `"O"`를 틀리다면 `"X"`를 순서대로 담은 배열을 return하도록 solution 함수를 완성해주세요.
## 풀이
- `quiz`에 들어오는 문자열의 내용을 공백으로 구분할 수 있기에 string.Split()을 이용하여 공백 기준으로 문자열을 나눴다.
- 문제에서는 덧셈 또는 뺄셈 밖에 없기에 삼항 연산자로 식들을 계산한 후 결과 또한 삼항 연산자로 비교하여 O/X를 구분했다.

``` C#
using System;

public class Solution {
    public string[] solution(string[] quiz) {
        // 문제마다 정답을 지정해야 하므로 quiz의 길이만큼 정답 배열 생성
        string[] answer = new string[quiz.Length];
        
        for(int i = 0; i < quiz.Length; i++)
        {
            // 공백을 기준으로 string 나눠서 저장하기
            string[] problem = quiz[i].Split(" ");
            
            // 공백 기준으로 나누었기에 각 인덱스에 맞는 숫자 저장 가능
            int num1 = int.Parse(problem[0]);
            int num2 = int.Parse(problem[2]);
            int equal = int.Parse(problem[4]);
            
            // 덧셈 또는 뺄셈 밖에 없기에 삼항 연산자로 연산 결과 저장
            int result = 
                problem[1].Equals("+") ? (num1+num2) : (num1-num2);
            
            // 연산 결과와 문제의 정답 일치 확인
            answer[i] = (result == equal) ? "O" : "X";
        }
        
        return answer;
    }
}
```

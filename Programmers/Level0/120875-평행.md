# 평행
### [Lv.0 - 120875](https://school.programmers.co.kr/learn/courses/30/lessons/120875)

## 문제
점 네 개의 좌표를 담은 이차원 배열  `dots`가 다음과 같이 매개변수로 주어집니다.
`[[x1, y1], [x2, y2], [x3, y3], [x4, y4]]`
주어진 네 개의 점을 두 개씩 이었을 때, 두 직선이 평행이 되는 경우가 있으면 `1`을 없으면 `0`을 return 하도록 solution 함수를 완성해보세요.
## 풀이
- 두 직선이 평행한 조건은 몇 가지 있지만 문제에서의 평행 조건만 보자면 직선의 기울기만 같으면 된다.
- 기울기 공식(각각 x와 y의 증가량이라 부른다) : $(y_2 - y_1) \div (x_2 - x_1)$
- 점이 얼마 없어 하나씩 비교하면서 적었지만 점이 여러개일 경우 반복문이나 Linq의 Distinct() 같은 중복을 잘 확인해야 한다.

``` C#
using System;

public class Solution {
    public int solution(int[,] dots) 
    {
        // 보기 편하게 점 4개를 따로 구분
        int[] dot1 = {dots[0,0], dots[0,1]};
        int[] dot2 = {dots[1,0], dots[1,1]};
        int[] dot3 = {dots[2,0], dots[2,1]};
        int[] dot4 = {dots[3,0], dots[3,1]};
        
        // 각각 기울기 비교
        if(Slope(dot1, dot2) == Slope(dot3, dot4))
            return 1;
        else if(Slope(dot1, dot3) == Slope(dot2, dot4))
            return 1;
        else if(Slope(dot1, dot4) == Slope(dot2, dot3))
            return 1;
        else
            return 0;
    }
    
    // 기울기 반환 함수
    double Slope(int[] a, int[] b) 
    {
        return (double)(b[1] - a[1]) / (b[0] - a[0]);
    }
    
}
```

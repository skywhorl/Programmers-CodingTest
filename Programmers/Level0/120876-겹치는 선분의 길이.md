# 겹치는 선분의 길이
### [Lv.0 - 120876](https://school.programmers.co.kr/learn/courses/30/lessons/120876)

## 문제
선분 3개가 평행하게 놓여 있습니다.
세 선분의 시작과 끝 좌표가 `[[start, end], [start, end], [start, end]]` 형태로 들어있는 2차원 배열 `lines`가 매개변수로 주어질 때, 두 개 이상의 선분이 겹치는 부분의 길이를 return 하도록 solution 함수를 완성해보세요.

## 풀이
- 선분이 겹치는 걸 확인하려면 비교하려는 두 점의`각 시작 지점 중의 최댓값 < 각 종료 지점 중의 최솟값`을 비교하면 된다.
- 위의 작업은 두 선분마다의 중복되는 구간만 구하는 것이기에 `[0, 5], [3, 9], [1, 10]` 처럼 `[1, 9]` 구간에서 두 개 이상의 선분이 겹쳐있으므로 8을 return 해야 하는 상황이 발생하기에 중복되는 구간의 길이를 제거해야 한다.
- 중복되는 구간을 제거하기 위해 아래의 코드에서는 중복되는 값이 안 들어가는 `HashSet` 컬렉션 클래스를 사용하였고 시작과 끝 지점을 기록하게 설정했다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int solution(int[,] lines) {
        // 중복된 선분을 거르기 위한 HashSet
        HashSet<int> overlap = new HashSet<int>();
        
        for(int i = 0; i < lines.GetLength(0); i++)
        {
            for(int j = i+1; j < lines.GetLength(0); j++)
            {
                // 앞의 점 최댓값
                int start = Math.Max(lines[i,0], lines[j,0]);
                // 뒤의 점 최솟값
                int end = Math.Min(lines[i,1], lines[j,1]);
                
                // 겹친다는 가정 하에 start의 좌표가 end 좌표 수보다 작아야 한다
                if(start < end)
                {                  
                    for(int k = start; k < end; k++)
                    {
                        overlap.Add(k);
                    }
                }
            }
        }
        
        // 겹치는 선분 길이
        return overlap.Count;
    }
}
```

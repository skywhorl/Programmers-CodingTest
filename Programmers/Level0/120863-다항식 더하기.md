# 다항식 더하기
### [Lv.0 - 120863](https://school.programmers.co.kr/learn/courses/30/lessons/120863)

## 문제
한 개 이상의 항의 합으로 이루어진 식을 다항식이라고 합니다. 
다항식을 계산할 때는 동류항끼리 계산해 정리합니다. 
덧셈으로 이루어진 다항식 `polynomial`이 매개변수로 주어질 때, 동류항끼리 더한 결괏값을 문자열로 return 하도록 solution 함수를 완성해보세요. 같은 식이라면 가장 짧은 수식을 return 합니다.
## 풀이
- x는 어디까지나 문자이기에 `Replace()` 함수를 통해 다른 걸로 대체해 수를 각각 계산하게 설계했다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public string solution(string polynomial) {
        // + 계산만 있으니 연산자와 공백 단위로 나누기
        string[] poly = polynomial.Split(new string[] {" + "}, StringSplitOptions.None);
        
        // 각각 x와 상수의 합을 구할 변수
        int xSum = 0;
        int numSum = 0;
        
        foreach(string formula in poly)
        {
            // x가 들어가 있을 경우
            if(formula.Contains("x"))
            {
                // 숫자만 계산하기 위해 x 빼두기
                string str = formula.Replace("x", "");
                // x는 1x이므로 1 카운트 하기
                if(str.Equals(""))
                {
                    xSum += 1;
                }
                else
                {
                    // x끼리의 합
                    xSum += int.Parse(str);
                }
            }
            else
            {
                // 숫자들끼리의 합
                numSum += int.Parse(formula);
            }
        }
        
        // 각 식의 결과에 맞게 저장
        List<string> answer = new List<string>();
        if(xSum != 0)
        {
            answer.Add(xSum == 1 ? "x" : xSum + "x" );
        }
        if(numSum != 0)
        {
            answer.Add(numSum.ToString());
        }

        // 항이 두 개면 자동으로 + 붙이기
        return string.Join(" + ", answer);
    }
}
```

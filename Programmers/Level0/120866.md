# 안전지대
### [Lv.0 - 120866](https://school.programmers.co.kr/learn/courses/30/lessons/120866)

## 문제
- 지뢰는 2차원 배열 `board`에 1로 표시되어 있고 `board`에는 지뢰가 매설 된 지역 1과, 지뢰가 없는 지역 0만 존재합니다.
- 지뢰가 매설된 지역의 지도 `board`가 매개변수로 주어질 때, 안전한 지역의 칸 수를 return하도록 solution 함수를 완성해주세요.
## 풀이
- 지뢰를 발견하였을 때 그 근처를 다시 배회해야 하기에 중첩 반복문이 많아 아래 코드에서는 9칸(발견 위치 포함)의 수색 범위를 미리 배열로 만들었다.
- 또한 배회 수색 범위가 `board`의 범위를 넘어서면 안 되며 지뢰마다의 위험 범위가 중복될 수 있기에 따로 위험 구역 배열을 만든 후 따로 다시 검사했다.

``` C#
using System;

public class Solution {
    public int solution(int[,] board) {
        int answer = 0;
        // 보드 길이
        int length = board.GetLength(0);

        // 지뢰와 근처 위험지대를 표시할 새로운 배열 
        int[,] dangerZone = new int[length,length];
        // 지뢰 근처 위험지대
        int[] row = { -1, -1, -1, 0, 0, 0, 1, 1, 1 }; 
        int[] column = { -1, 0, 1, -1, 0, 1, -1, 0, 1 }; 
        
        for(int i = 0; i < length; i++)
        {
            for(int j = 0; j < length; j++)
            {
                // 지뢰 발견
                if(board[i,j] == 1)
                {
                    // 배열로 미리 정해둔 구역으로 지뢰 근처 수색
                    for (int k = 0; k < 9; k++) 
                    {
                        int r = i + row[k]; 
                        int c = j + column[k]; 

                        // 수색 범위가 지도 안쪽인지 확인
                        if (r >= 0 && r < length && c >= 0 && c < length) 
                        {
                            // 위험지대 최종 수정(중복이 있을 수 있지만 1로 변경됨)
                            dangerZone[r,c] = 1;
                        }
                    }
                }
            }
        }
        
        // 위험지대 수색 후 0인 곳을 고르기
        foreach(int safe in dangerZone)
        {
            answer += (safe == 0) ? 1 : 0;
        }
        
        
        return answer;
    }    
}
```

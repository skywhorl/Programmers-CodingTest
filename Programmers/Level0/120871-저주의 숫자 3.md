# 저주의 숫자 3
### [Lv.0 - 120871](https://school.programmers.co.kr/learn/courses/30/lessons/120871)

## 문제
3x 마을 사람들은 3을 저주의 숫자라고 생각하기 때문에 3의 배수와 숫자 3을 사용하지 않습니다.
정수 `n`이 매개변수로 주어질 때, `n`을 3x 마을에서 사용하는 숫자로 바꿔 return하도록 solution 함수를 완성해주세요.
## 풀이
- 3의 배수는 나머지 계산으로 찾고 숫자 3을 포함하는 부분은 문자열로 바꿔 `Contains()` 함수를 이용해 찾았다.

``` C#
using System;

public class Solution {
    public int solution(int n) {
        int answer = 0;
        
        for(int i = 1; i <= n; i++)
        {
            // 기존 카운트
            answer++;
            
            // 3의 배수 또는 3을 포함하는 숫자일 때
            while(answer % 3 == 0 || answer.ToString().Contains("3"))
            {
                // 계속 다음 숫자로 넘기기
                answer++;
            }
        }
        
        return answer;
    }
    
}
```

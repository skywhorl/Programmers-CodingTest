# 배열 만들기 2
### [Lv.0 - 181921](https://school.programmers.co.kr/learn/courses/30/lessons/181921)

## 문제
- 정수 `l`과 `r`이 주어졌을 때, l 이상 `r`이하의 정수 중에서 숫자 "0"과 "5"로만 이루어진 모든 정수를 오름차순으로 저장한 배열을 return 하는 solution 함수를 완성해 주세요.
- 만약 그러한 정수가 없다면, `-1`이 담긴 배열을 return 합니다.
## 풀이
- 각 자릿수의 숫자를 확인하기 위해서 char 타입의 리스트에 담아 일일이 비교했다.
- 여기서, '52'이나 '25'처럼 일부만 포함될 수 있기 때문에 이 부분 또한 잘 확인할 필요가 있다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(int l, int r) {
        // 정답 내용이 들어갈 리스트
        List<int> answer = new List<int>();
        // 0 또는 5가 적혀 있는지 비교를 위한 리스트
        List<char> checkNum = new List<char>();
        // 0 또는 5만 들어가 있는지 확인하기 위한 변수 
        bool isAnswer = false;
        
        
        // l 이상 r 이하의 정수
        for(int i = l; i <= r; i++)
        {
            // 숫자를 문자열 -> 문자 리스트로 변환하여 담기
            string str = i.ToString();
            foreach(char chr in str)
            {
                checkNum.Add(chr);
            }
            // 문자 리스트마다 숫자 체크
            foreach(char num in checkNum)
            {
                // 0 또는 5가 존재한다면
                if(num.Equals('0') || num.Equals('5'))
                {
                    // 일단 정답으로 인정
                    isAnswer = true;
                }
                else
                {
                    // 0 또는 5가 존재하지 않기에 더 확인해 볼 필요 없이 break
                    isAnswer = false;
                    break;
                }
            }
            // 0 또는 5만 존재하면
            if(isAnswer)
            {
                // 정답 내용 추가
                answer.Add(i);
            }
            // 다음 문자 배열을 받기 위해 비워두기
            checkNum.Clear();
        }
        
        // 정답에 아무것도 안 들어가 있을 경우
        if(answer.Count == 0)
        {
            answer.Add(-1);
        }
        
        return answer.ToArray();
    }
}
```

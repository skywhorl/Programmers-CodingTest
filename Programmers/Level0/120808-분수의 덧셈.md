# 분수의 덧셈
### [Lv.0 - 120808](https://school.programmers.co.kr/learn/courses/30/lessons/120808)

## 문제
첫 번째 분수의 분자와 분모를 뜻하는 `numer1`, `denom1`, 두 번째 분수의 분자와 분모를 뜻하는 `numer2`, `denom2`가 매개변수로 주어집니다.
두 분수를 더한 값을 기약 분수로 나타냈을 때 분자와 분모를 순서대로 담은 배열을 return 하도록 solution 함수를 완성해보세요.
- 제한사항 : 0 < `numer1`, `denom1`, `numer2`, `denom2` < 1,000
## 풀이
- 기약분수 : 분모와 분자의 공약수가 1뿐이어서 1외에는 서로 나눌 수 없는 상태의 분수
- 기약분수 조건에 맞추려면 공약수가 1이 될 때까지 서로를 1외의 다른 수로 나눠야 한다([유클리드 호제법](https://github.com/skywhorl/Algorithm-CodingTest/blob/main/Programmers/Level0/120878-%EC%9C%A0%ED%95%9C%EC%86%8C%EC%88%98%20%ED%8C%90%EB%B3%84%ED%95%98%EA%B8%B0.md)).

``` C#
using System;

public class Solution {
    public int[] solution(int numer1, int denom1, int numer2, int denom2) {
        
        // 분수의 합 구하기
        int sumNumer = (numer1*denom2) + (numer2*denom1);
        int sumDenom = denom1*denom2;
        
        // 최대 공약수를 구하기 위한 변수
        int numer = sumNumer;
        int denom = sumDenom;
        int remainder = 0;
        
        // 나머지가 0이 될 때까지 서로 나눠서 최대 공약수 구하기
        while(denom != 0)
        {
            // 서로를 나눈 나머지 구하기
            remainder = numer % denom;
            // 나머지 계산 갱신
            numer = denom;
            denom = remainder;
        }
        
        // numer : 구한 최대 공약수
        return new int[] {sumNumer / numer, sumDenom / numer};
    }
}
```

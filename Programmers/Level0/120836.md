# 순서쌍의 개수
### [Lv.0 - 120836](https://school.programmers.co.kr/learn/courses/30/lessons/120836)

## 문제
- 순서쌍이란 두 개의 숫자를 순서를 정하여 짝지어 나타낸 쌍으로 (a, b)로 표기합니다.
- 자연수 `n`이 매개변수로 주어질 때 두 숫자의 곱이 `n`인 자연수 순서쌍의 개수를 return하도록 solution 함수를 완성해주세요.
## 풀이
- 순서쌍을 직관적으로 표현하자면, 1부터 자연수 n까지 전부 비교해 n을 나누었을 때 나머지가 0이면 그 몫과 수는 순서쌍이 된다.

``` C#
using System;

public class Solution {
    public int solution(int n) {
        int answer = 0;
        
        // 1부터 n까지 순서쌍 비교하기
        for(int i = 1; i <= n; i++)
        {
            // n이 i로 나누어 떨어지면 나눈 몫과 i는 순서쌍
            if(n%i == 0)
                answer++;
        }
        return answer;
    }
}
```
